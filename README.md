Jeu de calcul mental Android Studio

Concept : Des calculs vont etre affiché et devront être résolu en un temps limité par l'utilisateur.

Menu :
- Mode classique : 30 niveaux, de plus en plus difficiles. Le joueur dispo de 3 vies. A la fin du mode, le joueur est récompensé par une étoile sur son profil et accède au mode difficile.
- Mode difficle : 30 niveaux, toutes les opérations sont présentes dès le début. Le joueur est récompensé par une deuxieme étoile a la fin du mode.
- Mode endless : nombre de niveaux infinis, une étoile sera octroyé au joueur s'il parvient au niveau 48. Le joueur peut regagner une vie, pour un maximum de 4, en battant un calcul plus difficile appelé "boss".
- Highscores : Tableau affichant le score des 10 meilleurs joueurs.
